----------------------------------------------- Pre Entrga ------------------------------------------------

Este es mi proyecto de una página web básica desarrollada como parte de la Pre-Entrega del curso Front-End. Mi tematica es jugueteria por ninguna razon en particular, simplemente me gusto la idea. Empeze entrando en varias paginas de jugueterias para sacar ideas de como estructurarla y de contenido.
Encare el proyecto de arriba hacia abajo, la página está estructurada con HTML semántico y utiliza las etiquetas <header>, <main>, y <footer> para organizar el contenido. Lo primero fue agregar la metadata en el <head>. Despues en el <header> arranque con la barra de navegacion, utilice flexbox para distribuir los elementos, me costo al principio pero mientras mas lo usaba y probaba mas le iba sacando la onda. El banner y el logo le pedi a la ia que me los cree, uso copilot.
Segui con el <main>, titulo y tarjetas, me sirvio los ejemplos de los after de las tarjetas para hacerlas a mi gusto, el resto de las imagenes las baje buscando por google. Termine usando varias familias tipograficas, queria usar solo un par pero para probar las distintas formas de usarlas termine con 7 tipografias distintas. Use google fonts, y el resto las descargue de dafont.com.
El resto del <main> es la publicidad de Mario, y los personajes de los muñecos que los acomode con wrap al ser varios recuadros y  que se desbordaban. El newsletter lo iba incluir en el footer pero me gusto como quedo en la practina de la pre entrega del after y lo cambie. Y el <foter> combine varios elementos de informacion que normalmente se ponen, redes, contactos, preguntas, etc. 
Inclui la pagina de contacto con el formulario, bastante simple de hacer. El diseño estoy conforme pero obviamente me falta practica y herramientas para mejorar, flexbox es lo que mas tiempo me llevo pero las media queries siento que fue lo mas dificil para aplicarlas y no termine de aplicarlas bien del todo creo. Todavia las estoy practicando y retocando. Me sirvieron mucho los ejemplos en los after, el material de clase y despues copilot lo consulte con propiedades que no coonocia como las pseudo clases, o comandos de diseño como letter-spacing, text-transform, outline, line-height, etc,etc. Son muchas las herramientas de diseño y con practicando las voy conociendo.

----------------------------------------------- Entrga ------------------------------------------------

Para la entrega agregue en el html, 3 banners, y mas tarjetas de productos con el script para que se puedan agregar al carrito. Utilice bootstrap con el carousel de imagenes en el banner para agregar mas imagenes, ademas de bootstrap alerta en tienda para indicar que se agrego un producto al carrito. En contacto valide los datos y agregue una alerta para indicar que se envio correctamente. En tienda no consegui una Api para consumir, encontre 2 o 3 de jugueterias pero no las pude hacer andar. No se si estaban caidas o que tenian, escribi una array como los ejemplos que diste en clases pero despues encontre una pagina llamada https://mockapi.io/, es de paga al parecer pero te da de gratis una cierta cantidad de proyectos. Escribi los array ahi y pude consumirlo desde ahi, y como ya tenia la lista escrita se me ocurrio combinarlas.
