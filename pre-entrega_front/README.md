# 🎲 Juguetería Fantasía

Proyecto web para una tienda ficticia de juguetes llamada **Juguetería Fantasía**.  
El sitio está desarrollado en **HTML y CSS**, con un diseño moderno, colorido y responsivo.

---

## 📑 Descripción

Este proyecto corresponde a una **pre-entrega** de un sitio web de e-commerce.  
Incluye:

- **Header** con logo, barra de navegación, buscador y accesos rápidos (login, favoritos, carrito).
- **Banner principal** con imagen destacada.
- **Sección de ofertas** con tarjetas de productos en promoción.
- **Publicidad** con imagen destacada.
- **Catálogo de personajes** (DC, Marvel, Dragon Ball, Disney, etc.).
- **Newsletter** para suscripción de usuarios.
- **Footer** con información de contacto, enlaces útiles y redes sociales.

---

## 🛠️ Tecnologías utilizadas

- **[HTML5](ca://s?q=Aprender_HTML5)** para la estructura del sitio.
- **[CSS3](ca://s?q=Aprender_CSS3)** para estilos y diseño responsivo.
- Imágenes en formatos `.jpg`, `.png`, `.webp`, `.avif`.
